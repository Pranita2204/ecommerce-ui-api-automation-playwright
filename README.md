# Playwright Java E-Commerce Automation Framework

This project is a UI automation framework built using Playwright with Java.

## Tech Stack
- Java
- Playwright
- TestNG
- Maven

## Features
- Page Object Model
- Login Automation
- Add to Cart Flow
- TestNG Execution
- Cross-browser support

## How to Run
mvn clean test
